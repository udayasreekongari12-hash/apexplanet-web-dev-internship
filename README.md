# ApexPlanet Software – Web Development Internship

## 📌 About This Internship
This repository documents my journey through a Web Development Internship 
at **ApexPlanet Software Pvt. Ltd.**, covering the core technologies of 
front-end development — **HTML, CSS, and JavaScript**.

The internship is structured into 5 progressive tasks, starting from the 
basics of web pages and moving toward building a fully responsive, 
interactive web project. Each task folder contains the source code, 
along with a brief explanation of the objectives and concepts applied.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 Tasks Overview
| Task | Title | Focus Area |
|------|-------|------------|
| Task 1 | Basics of Web Development | HTML structure, CSS styling, basic JS interactivity |
| Task 2 | Intermediate HTML, CSS & JS | Forms, validation, Flexbox/Grid, DOM manipulation |
| Task 3 | Advanced Styling and JavaScript | Responsive design, interactive quiz/carousel |
| Task 4 | Full Project Implementation | End-to-end project build |
| Task 5 | Final Project and Optimization | Performance & final polish |

## 🎯 Goal
To strengthen my front-end development skills through hands-on, 
task-based learning under real-world project guidance.
